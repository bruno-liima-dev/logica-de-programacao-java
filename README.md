# Lógica de Programação em Java

Este repositório contém exercícios e exemplos desenvolvidos em Java,
baseados no curso de Lógica de Programação que estou estudando na Udemy.

## Conteúdos
- Variáveis e tipos de dados
- Entrada e saída de dados
- Estruturas condicionais
- Laços de repetição
- Vetores e matrizes

## Objetivo
Construir uma base sólida em programação, iniciando pela lógica com Java,
e evoluir progressivamente para o desenvolvimento backend e fullstack.
